Политика Конфиденциальности для приложения CashBackCheck

1. Введение

Данное приложение CashBackCheck (далее "Приложение") разработано и поддерживается частным лицом под именем GoFlow. В целях защиты конфиденциальности пользователей и обеспечения прозрачности использования их данных, настоящая Политика Конфиденциальности (далее "Политика") объясняет, какие данные собираются, как они используются и защищаются.

2. Сбор данных

Приложение CashBackCheck собирает следующие типы данных:

  • Номер телефона: Используется для отправки проверочного кода, необходимого для входа в Приложение.
  • Информация о придуманных пользователем названиях карт и суммы операций по ним: Пользователь вручную вводит удобные названия карт и операции, которые используются для расчета накопленного кешбэка.

Приложение не интегрировано с какими-либо внешними сервисами или SDK для сбора дополнительных данных.

3. Использование данных

Собранные данные используются для следующих целей:

  • Восстановление доступа к учетной записи пользователя через проверочный код, отправленный по SMS.
  • Обеспечение корректного расчета кешбэка на основе данных, введенных пользователем.
  • Поддержка пользователей по вопросам работы Приложения.

4. Обмен данными

Приложение CashBackCheck не передает данные пользователей третьим лицам.

5. Защита данных

Для защиты данных пользователей Приложение использует метод аутентификации через SMS-код. Только после успешного ввода кода доступ к авторизованной зоне Приложения будет предоставлен.

6. Хранение данных

Данные пользователей, введенные в Приложение, не имеют срока давности и хранятся до тех пор, пока пользователь не решит их удалить.

Удаление данных: Пользователь может самостоятельно удалить все свои данные, включая информацию о картах, категориях и операциях, используя соответствующие функции Приложения.

7. Права пользователя

Пользователь имеет следующие права в отношении своих данных:

  • Право на доступ: Пользователь может в любое время просматривать введенные данные в Приложении.
  • Право на изменение: Пользователь может изменять данные о картах и операциях.
  • Право на удаление: Пользователь может удалить все свои данные через функционал Приложения.
